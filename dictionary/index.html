Dictionary

Here, you can see all of Baslamo's vocabulary in one central place! You can search a Baslamo or English word in the input box, or search by a specific field by using @field term. This input uses RegEx, so you can use things like ^$ for empty strings or . for any text.

The fields are always all-lowercase, and include xx_def, xx_notes, xx_tags (replace xx with a language code, e.g. en for English), etymology, etym_tags, and derive.

Click on an entry to expand it. If you don't like looking at all the empty fields, you can select "All non-empty" in the dropdown next to the input box.
