# Lesson 3 — Making new words

Baslamo's vocabulary has many terms made up of multiple words. While sometimes these compound words are made of two full words, most of the time they're made up of one full word and one or more affixes, which in Baslamo are called _kesekmolas_. We'll look at an overview of these affixes, then go into each one in a bit more detail.

| _kesekmol_ | Meaning             | Examples                                   |
|:----------:|:--------------------|:-------------------------------------------|
| -asu       | group, collection   | _mot_ "tree" → _motasu_ "forest"           |
| -tel       | do-er, occupation   | _fas_ "to create" → _fastel_ "creator"     |
| -igi       | causing, result     | _al_ "knowledge" → _aligi_ "to teach"      |
| -ik        | little, small       | _siku_ "ice" → _sikunik_ "snow"            |
| -lo        | tool, item          | _naso_ "to write" → _nasolo_ "pencil, pen" |
| -meti      | place for something | _vangu_ "to buy" → _vangumeti_ "store"     |
| -pi        | becoming, changing  | _al_ "knowledge" → _alpi_ "to learn"       |
| -sini      | being, state, -ness | _vaj_ "fast" → _vajsini_ "speed, velocity" |
